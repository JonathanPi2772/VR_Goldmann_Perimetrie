# Perimetrie mittels VR

Dies ist ein Reasearch Project im Rahmen des Studiums in der Medizinischen Informatik. 

Es handelt sich um eine Implementierung einer kinetischen Geischtsfelduntersuchung nach Goldmann für eine Vive Focus 3 mit Eye Tracker. 

Alle Ergebnisse finden sich im Paper. Die Messergebnisse sind auch verfügbar. 

Im Code findet sich unter waves/hellovr (Adaption von dem Wave Beispiel Project) die Application für die VR Brille. 
Ansonsten sind noch Python Prototypen zu finden, die aus den ersten Entwicklungswochen stammen.
